## 🚀 How to Use

1. **Find your lab** — Browse the folders. Each folder matches a lab name.
2. **Open Cloud Shell** — In Google Cloud Console, activate Cloud Shell.
3. **Run the script** — Copy the `curl` command from the `TechCode.md` inside the lab folder.

**Example:**
```bash
# Set required variables first (check individual lab folder for exact vars needed)
export REGION=us-central1

# Then run the lab script
curl -LO raw.githubusercontent.com/VELU1231/Google-Cloud-Arcade-Masterclass/refs/heads/main/[Lab-Folder-Name]/TechCode.sh
sudo chmod +x TechCode.sh
./TechCode.sh
```
